# 06 - Video Demo Guide (5-10 phut)

## Muc tieu video
Demo day du 3 phan: Selenium, NUnit logic, JMeter.

## Timeline de xuat
1. Gioi thieu bai toan (30-45 giay)
2. Demo Selenium UI tests (2-3 phut)
3. Demo NUnit logic tests (1-2 phut)
4. Demo JMeter run va bao cao (2-3 phut)
5. Tong ket ket qua + bug noi bat (30-60 giay)

## Script thao tac
1. Mo project va gioi thieu cau truc thu muc nop bai.
2. Chay `dotnet test` de show ket qua Selenium + NUnit.
3. Mo `SearchLogicTests.cs` giai thich 2 case keyword rong/keyword dai.
4. Mo JMeter, run file `jmeter/NewsSites-Stress-500-1000.jmx`.
5. Show Summary Report + Graph Results.
6. Ket luan ve do on dinh va de xuat cai tien.

## Luu y quay video
- Do phan giai 1080p, font de doc.
- Hien ro command va output pass/fail.
- Chup canh Summary Report va Graph Results truoc khi ket thuc video.
